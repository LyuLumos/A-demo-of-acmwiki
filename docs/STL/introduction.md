## 什么是STL？
STL（Standard Template Library），即标准模板库，是一个具有工业强度的，高效的C++程序库。
STL 借助模板实现了常用的数据结构及其算法，并且做到了数据结构和算法的分离。


STL包括容器（Container）
迭代器（Iterator）
算法（Algorithm）
仿函数（Functor）
适配器（Adaptor）
分配器（Allocator）六大组件，在算法竞赛里有广泛的应用。
